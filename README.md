# SOC Threat Hunting Labs

Welcome to my Threat Hunting portfolio.

This repository documents my hands-on journey in threat hunting, detection engineering, and security investigations. Each lab is designed to simulate real SOC workflows using industry-standard tools and frameworks.

## Technologies

- Sysmon
- Splunk
- Sigma Rules
- MITRE ATT&CK
- Windows Event Logs
- PowerShell
- Wireshark

## Objectives

- Build endpoint visibility.
- Investigate suspicious activity.
- Create and test detection rules.
- Map findings to MITRE ATT&CK.
- Improve threat hunting and incident response skills.
